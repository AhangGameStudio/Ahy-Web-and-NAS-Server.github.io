# 云NAS存储系统

一个基于Web的NAS（网络附加存储）系统，允许用户上传、存储和管理各种类型的文件，包括视频、图片、音频和文档等。

**访问地址：[www.ahangya.cyou](http://www.ahangya.cyou)**

## 功能特点

- 📤 **文件上传**：支持拖拽上传和点击选择文件
- 📂 **文件管理**：浏览、下载和删除已上传的文件
- 🎨 **文件预览**：图片文件可直接预览，其他文件显示类型图标
- 🔍 **文件分类**：按文件类型（图片、视频、音频、文档）过滤查看
- 📱 **响应式设计**：适配桌面端和移动端设备
- 💾 **本地存储**：使用浏览器本地存储保存文件信息

## 支持的文件类型

- **图片**：JPG, PNG, GIF, WEBP等
- **视频**：MP4, AVI, MOV, WMV等
- **音频**：MP3, WAV, FLAC, OGG等
- **文档**：PDF, DOC, DOCX, TXT, ZIP等

## 特殊功能

### NCM格式音乐文件处理
- 自动检测NCM格式文件（网易云音乐专有格式）
- 模拟转换为MP3格式（注意：由于浏览器安全限制，纯前端无法实现真正的NCM解密）
- 仅在本地开发环境中可完整体验此功能
- 在GitHub Pages等静态托管服务上，NCM文件将作为普通文件处理

## 使用方法

1. 打开 `index.html` 文件
2. 点击上传区域或拖拽文件到上传区域
3. 点击"开始上传"按钮上传文件
4. 上传后的文件会显示在"我的文件"区域（逻辑上存储在"Save-Fill"文件夹中）
5. 可以通过顶部的过滤器按类型查看文件
6. 点击文件卡片上的"下载"按钮下载文件
7. 点击"删除"按钮删除文件

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage (用于本地数据存储)

## 注意事项

- 文件存储在浏览器本地存储中，清除浏览器数据会导致文件丢失
- 不同浏览器对本地存储的容量限制不同
- 这是一个前端演示项目，不涉及后端服务器
- 上传的文件逻辑上存储在"Save-Fill"文件夹中

## 项目结构
